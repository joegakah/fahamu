# **App Name**: Fahamu

## Core Features:

- AI-Powered Weakness Identification: Use an LLM to analyze student performance data across subjects and topics to pinpoint specific areas of academic weakness. Use tools to make judgments on inclusion/exclusion of different indicators. Input data can be manually uploaded by students or integrated via API from school databases.
- Personalized Study Plan Generation: Automatically generate study timetables tailored to individual student needs, focusing on weakness improvement, strength enhancement, and upcoming exams.
- Study Method Guidance: Provide students with a curated selection of effective study methods and resources applicable to their specific learning challenges and strengths. Pull content using reasoning based on the context and their existing capabilities, when appropriate
- Data Visualization Dashboard (Students): Visually represent student performance data, highlighting areas of improvement and overall progress.
- Student Performance Overview (Teachers): Provide teachers with aggregated student performance data to identify common areas where additional instructional focus is needed.
- Resource Allocation Insights (Administrators): Offer school administrators insights into student learning outcomes to inform resource allocation and identify areas for curriculum improvement.
- Data Input and Management: Allow manual input of student data and enable integration with existing school databases via API.
- Virtual Classes: Enable students to be organized in classes, where subject teachers and class teachers can see their students and see trends in performance per subject per topic and help then create lesson plans

## Style Guidelines:

- Primary color: Teal (#008080) to represent growth and intellect.
- Background color: Very light grey (#F0F0F0), close to white, to provide a clean and non-distracting backdrop.
- Accent color: Pale Yellow (#D3D391) to highlight insights.
- Body and headline font: 'PT Sans', a versatile sans-serif font, is suitable for both headlines and body text. 'PT Sans' offers a modern, yet warm aesthetic for educational content.
- Use clear, academic-themed icons (e.g., books, charts, graphs) to represent data and insights.
- Design a clean and intuitive dashboard layout with clear data visualizations and easy navigation.
- Incorporate subtle animations (e.g., data loading, progress updates) to enhance user engagement.
